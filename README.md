# Hospital CRM System

A comprehensive, modern Hospital Management System built with Next.js, TypeScript, and PostgreSQL.

## Project Structure

```
hospital-crm/
├── client/          # Next.js frontend
├── server/          # Express.js backend
├── shared/          # Shared types and utilities
└── docker/          # Docker configuration
```

## Features Overview

### Core Modules
1. Patient Management System
2. Doctor & Staff Management
3. Appointment Scheduling
4. Electronic Health Records (EHR)
5. Billing & Insurance Claims
6. Pharmacy Management
7. Laboratory Information System
8. Radiology & Imaging
9. Inventory & Asset Management
10. Analytics & Reporting Dashboard

### Advanced Features
- Real-time notifications
- Telemedicine integration
- AI-powered diagnosis assistance
- Multi-language support
- Mobile responsive design
- Role-based access control (RBAC)
- Audit trails & compliance
- Emergency management system

## Technology Stack

### Frontend
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Zustand (State Management)
- React Query (Data Fetching)
- Chart.js (Analytics)

### Backend
- Node.js + Express.js
- TypeScript
- PostgreSQL (Database)
- Prisma ORM
- JWT Authentication
- Socket.io (Real-time)
- Redis (Caching)

### DevOps
- Docker & Docker Compose
- ESLint & Prettier
- Git

## Getting Started

### Prerequisites

- PostgreSQL 14+
- Docker (optional)
